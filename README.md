ATC — десктоп-приложение (Java Swing)

Программа для управления междугородними тарифами «АТС». GUI-порт консольной ЛР-3: добавление/изменение/удаление тарифов, импорт/экспорт CSV, сортировка, расчёты.

Функциональные возможности

Авторизация (опционально, если нужно — заглушка).

Тарифы:

добавление / редактирование / удаление;

отображение списка в таблице (JTable, сортировка по клику);

расчёты: средняя цена, общая сумма, массовое изменение цен на %;

импорт из CSV и экспорт в CSV (UTF-8 + BOM для Excel).

Проверки и устойчивость:

валидация города/цены/скидки в TariffManager;

предметные ошибки TariffException → диалоги с сообщениями;

блокировка поля «Скидка» для обычного тарифа.

UI (минимум 2 окна):

MainFrame — главное окно (таблица + меню);

TariffFormDialog — диалог добавления/редактирования
(перегрузка конструктора: режим «Добавить» / «Изменить»).

Стек и структура проекта

Java 17+, Swing (без JavaFX).

Аналоги требований: JTable (как DataGridView), JFileChooser (как OpenFileDialog).
